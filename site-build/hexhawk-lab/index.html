HexHawk Lab

Test carefully. Say only what was proven.

The lab is where HexHawk checks hard cases and records limits. The rule is simple: if a test did not prove it, the website and reports should not claim it.

Native proof matters

A browser preview is not the same as the real Windows app. Native app claims need native app proof.

Do not claim attacks that were not proven

HexHawk can show safety clues, but it should not claim a working attack or final proof unless that exact result was validated.

Keep old results honest

An old green test is history. Current claims need current proof.

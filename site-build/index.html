WHAT HEXHAWK DOES

From file input to reviewable output.

Open a binary on a controlled test machine. HexHawk shows hashes, file metadata, strings, imports, sections, code views, function evidence, uncertainty, and exportable notes. It is a workbench for analysts, not a promise that unsafe files become safe.

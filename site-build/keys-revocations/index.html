Keys & Revocations

Know which trust keys are allowed.

Keys help prove that a file, report, or trust record came from the expected source. A revoked key is a key that should no longer be trusted.

Allowed keys

Use current keys only for the purpose listed with them.

Revoked keys

If a key is revoked, stop trusting new material that depends on it.

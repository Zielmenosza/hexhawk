AetherFrameGuard

Measure first. Change one safe thing. Test again.

AetherFrameGuard is separate from HexHawk. It helps a player test CS2 and PC performance changes without guessing. It should show what was measured, what changed, and what still was not proven.

1. Measure

Run a short FPS and PC capture. Save what was detected, what was measured, and what failed to measure.

2. Try one safe change

If you ask it to, apply one conservative setting change that can be backed out and tested again.

3. Re-test

Compare the new run with the last run so you know whether the change helped, hurt, or stayed unclear.

Current caveat

Live FPS proof depends on CS2, PresentMon, and your PC at that moment. Logs should say exactly what was captured and what remains unknown.

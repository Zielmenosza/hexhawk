AetherFrameGuard

Measure first. Change one safe setting. Re-test.

AetherFrameGuard is separate from HexHawk and AETHERFRAME. It helps a player test CS2/PC performance changes with clearer logs and safer next-action guidance.

Measure

Run a bounded FPS/PC capture and record what was detected, what was measured, and what was not proven.

Apply one safe trial

When requested, apply one conservative setting change that can be reviewed and re-tested.

Re-test

Compare the next run against the last result instead of guessing whether a setting helped.

Current caveat

Live FPS proof depends on CS2, PresentMon, and the local PC state. Logs and diagnostics should say exactly what was captured and what remains unproven.

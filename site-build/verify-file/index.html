BROWSER VERIFY

Verify hash and detached signature locally in your browser.

No upload occurs. Files are processed in-browser with Web Crypto so analysts can verify integrity and signatures before opening artifacts.

Verification inputs

Registry Status: Not loaded

Computed SHA-256: (none)

Hash Match: NOT CHECKED

Signature Status: NOT CHECKED

Details: Provide a file, then compute hash or run full verification.

Automation endpoints

  • Key registry JSON: /trust/keys.json
  • Revocations JSON: /trust/revocations.json
  • Detached signatures JSON: /trust/signatures/latest/signatures.json
  • Signed timestamps JSON: /trust/signed-timestamps.json
  • Discovery JSON: /.well-known/hexhawk-trust.json

Recommended process

  • Download checksums from /downloads/checksums.txt.
  • Load trust endpoints and select a known key ID when possible.
  • Treat unknown or revoked keys as failed trust decisions.

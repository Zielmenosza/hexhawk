Verify a file

Make sure the file is the one you meant to download.

A hash is like a fingerprint for a file. If the SHA-256 hash matches the expected value, you likely have the right file. If it does not match, do not install it.

1. Download

Save the installer or file from the expected download page.

2. Compute SHA-256

Use your system tool to compute the file hash.

3. Compare

Match every letter and number. One different character means the file is not the expected one.

Hash check is not signing

A matching hash helps prove file identity. It does not prove the file is Authenticode-signed or approved by Windows.

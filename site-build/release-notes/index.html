Release notes

2026-06-20 unsigned deployment candidate and engineering update

HexHawk has a smoke-passed unsigned Windows deployment candidate for controlled testing, plus new GYRE SSA fallback hardening on main. Public release is still blocked until real signing, updater validation, and full exact-artifact export proof are complete.

What changed recently

  • June 20 deployment candidate: MSI extraction smoke passed and NSIS silent install smoke passed.
  • The NSIS installer includes the real `WebView2Loader.dll` payload needed for native launch.
  • The current downloadable MSI/NSIS hashes are listed on the Downloads page and in `downloads/checksums.txt`.
  • GYRE SSA construction now detects irreducible multi-entry control-flow cycles and falls back instead of emitting misleading SSA output.
  • Main has moved beyond the tested candidate with docs and SSA commits, so any new deployment-candidate tag requires a fresh full gate.

Known caveats

  • Not a public-trusted general-availability release.
  • Current MSI and NSIS installers are Authenticode `NotSigned`.
  • Expect Windows trust warnings until real code signing is configured and verified.
  • Updater metadata is not validated for the June 20 unsigned candidate.
  • Full Open → Inspect → NEST → Export parity was not rerun for a signed/public artifact.

Before you install

Use a controlled test machine. Verify hashes. Keep the package manifest. Do not disable Windows security globally. If Windows or endpoint security blocks the unsigned package, report the exact warning and wait for guidance or a signed build.

  • Good for: private testing, demos, workflow feedback, install feedback.
  • Not for: public download, company-wide rollout, production malware handling, or automatic updates.
  • Expected: Windows unknown-publisher or reputation warnings because the artifacts are unsigned.

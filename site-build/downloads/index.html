Before you install

Use a test machine or controlled environment. These installers are unsigned, so Windows may warn you. That warning is expected until HexHawk has public-trusted code signing.

  • Good for: controlled internal testing, demos, and technical diligence.
  • Not for: broad public rollout, procurement-ready deployment, or automatic updater distribution.
  • MSI extraction and NSIS install smoke passed on the June 20 candidate.
  • Full Open → Inspect → NEST → Export parity must be rerun before an external signed release.
  • Always verify the SHA-256 hash before installing.
